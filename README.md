# Home Loan Saver
Home Loan Saver is a Python-based application that helps users compare the cost difference between a normal home loan and an overdraft (OD) home loan. It calculates the impact of interest rate, tenure, and loan amount on the total repayment and savings. Built with Pandas and Math for backend and Stream-lit for an interactive user interface.
## Features
- Compare Normal Home Loan and OD Home Loan.
- Interactive input fields for loan amount, interest rate, tenure, and monthly payment.
- Fast backend computations using Pandas and Math packages.
- Clean and intuitive user interface built with Streamlit.
- Tabular representation of total payments, interest paid, and potential savings.
## Tech Stack
- Frontend: Streamlit  
- Backend: Python (Pandas, Math)
